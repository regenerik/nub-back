📌 API - Administración by QuanThink

Esta es una plantilla de API que debe ser modificada según las necesidades del proyecto. Abajo se listan las rutas funcionales. Es posible que algunas rutas necesiten Authorization en Headers al llamarse. Para obtener la Key, contacte con un administrador.

📌 Ruta de Prueba

GET /test_admin_bp

Respuesta de prueba para verificar que la API está funcionando.

📌 Documentación

GET /

Devuelve la documentación de la API. Exactamente donde estas parado ahora.

📌 Crear Usuario

POST /create_user

Crea un nuevo usuario en el sistema.

📌 Iniciar Sesión

POST /login

Genera un token de acceso para autenticación.

📌 Obtener Usuarios

GET /users (Requiere Token)

Lista todos los usuarios registrados.

📌 Actualizar Perfil

PUT /update_profile

Modifica los datos de un usuario registrado.

📌 Actualizar Imagen de Perfil

PUT /update_profile_image

Cambia la imagen de perfil del usuario.

📌 Cambiar Rol de Administrador

PUT /update_admin

Asigna o revoca permisos de administrador a un usuario.

📌 Obtener Usuario por número de identidad

GET /get_user/<dni>

Busca y devuelve los datos de un usuario por su número de identidad.
